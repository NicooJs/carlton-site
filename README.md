# CARLTON

Frontend do site **Carlton**.

---

## Sobre

Carlton é um **projeto audiovisual** que se manifesta em diferentes formatos.  
O site funciona como um espaço de apresentação e comercialização de objetos ligados ao projeto.

Este repositório contém o **frontend da aplicação**, responsável pela interface, organização visual e experiência de navegação.

---

## Abordagem

- Interface pensada como parte da linguagem visual  
- Organização editorial  
- Estrutura simples e modular  
- O comércio existe, mas não define o projeto

---

## Stack

- React  
- Vite  
- JavaScript  
- CSS
